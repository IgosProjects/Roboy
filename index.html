What is Roboy.lua

Roboy.lua is a roblox console that allows you to make any game for it and put them in roblox Roboy.lua is a easy tool to learn lua it is a nice tool for those who still remember the old days roboy.lua puts roblox to its limits with its no ending functions and visible code docs and support roboy.lua is made and supported by IgoCodesThings IgoCodesThings is a youtube channel run by Igor Seili Igor Seili is a web/game/etc developer who just appears to have made this page roboy.lua is a lets say like a game engine like Unity but its more of a verison of Scratch but text roboy.lua was originaly a school project but well it became a big big but big project that can do all sorts of things Roboy.lua is a free easy to use library for games in roblox Roboy.lua is a fun and easy way for you and your friends to play or make games maybe sometime we will have a multiplayer game for the RoBoy Roboy.lua is made for fun some games that are accepted by IgoCodesThings for the games you can check out our games
